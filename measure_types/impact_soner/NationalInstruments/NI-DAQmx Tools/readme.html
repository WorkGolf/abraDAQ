NI-DAQmx Tools for The MathWorks, Inc. MATLAB® Software Release 1.0 Readme

January 2006

This file contains important last-minute information about NI-DAQmx Tools.

Installation

  1. Create a temporary folder on your local hard drive.
  2. Extract the zip file into the folder created in Step 1.
  3. To launche the installer, run the steup.exe program created after completing Step 2.

Known Issues

  • NI-DAQmx Tools has been tested with versions 13 and 14 of The Mathworks, Inc. MATLAB® software.
  • After clicking any one of the options in the DAQ Library Help menu, the NI-DAQmx Tools Help loads but the panel on the right does not load the selected HTML document. Click any of the topics in the navigation tree in the left panel to correctly load the help content.
  • Launch_Examples_Help function is not supported in this release of NI-DAQmx Tools.
